# GDD: シャドウダッシュ

## コンセプト
- ジャンル: action / dodge
- コアループ: 自分の影が0.5秒遅れて追いかけてくる → スワイプで逃げる → 影が加速 → 限界に挑戦
- 対象: 反射神経を競いたいカジュアル〜ミドルコアプレイヤー
- プレイ時間: 1プレイ30秒〜3分

## MDA分析
- **Mechanics**: スワイプ操作で発光する円キャラを動かす。0.5秒遅延で「影」が同じ軌跡をトレース。影との接触で即ゲームオーバー。生存時間でスコア加算。
- **Dynamics**: プレイヤーは「動かないと安全」という直感に反し、移動しないと自分の位置に影が重なるため必ず動く必要がある。動けば動くほど影の軌跡が複雑化し、自分自身の動きが障害物になる。
- **Aesthetics**: 緊張、自己との対峙、フロー（時間が歪む感覚）、コントロールの限界へのチャレンジ。

## ゲームフロー
1. タイトル画面（ロゴ + タップで開始）
2. **チュートリアル**（初回 or `?` ボタン）
3. ゲームプレイ
4. ゲームオーバー（影に触れた瞬間）
5. リザルト（スコア共有・リトライ）

## 操作方法
- **スワイプ**（任意方向）: キャラクター移動
- **タップ**（キャラ位置）: ターボブースト（クールタイム3秒、距離+30%）
- PC版: マウスドラッグでも可

## ゲーム要素
1. **プレイヤー**: 発光する青色の円。スワイプで滑らかに移動。
2. **影**: 同色の暗いシルエット。0.5秒遅延でプレイヤーの軌跡を完全トレース。徐々に追跡速度が上昇。
3. **スコア**: 生存時間 × 1.0 + コンボボーナス（影と一定距離保持で倍率）。
4. **障害物**（10秒経過後）: 画面上をランダムに動く小さな赤い円。触れたらスコア半減（即死ではない）。
5. **コインアイテム**: 黄色の星。取得でスコア+50。
6. **画面シェイク**: 影が接近時にカメラが軽く揺れる。
7. **BGMテンポ上昇**: 影との距離が縮まると音楽が速くなる。
8. **トレースリプレイ**: ゲームオーバー時にプレイヤーと影の軌跡を線で表示。

## 難易度曲線
- 0-15秒: 影の追跡速度 = 0.6（楽勝、操作慣れ）
- 15-30秒: 影の追跡速度 = 0.75（緊張感）
- 30秒-: 影の速度がさらに 0.05/10秒 で上昇 + 障害物追加
- 60秒+: 障害物が複数（上級者向け）

## チュートリアル設計（必須）
### 教える内容
1. **基本操作**: 「画面をスワイプしてキャラを動かそう！」
2. **ゲーム目的**: 「影に追いつかれないように逃げ続けよう」
3. **失敗条件**: 「影に触れたらゲームオーバー！」
4. **コツ**: 「同じ軌跡を辿らないように円を描いて逃げよう」

### チュートリアル方式
- インタラクティブ形式（実際にスワイプさせる）
- 矢印アイコンで操作箇所をハイライト
- 各ステップは1文+操作で完結
- 2回目以降はスキップ可能（localStorage）

## サウンドデザイン
- BGM: 緊張感のあるシンセ音楽（影との距離でテンポ上昇）
- SE: スワイプ音、コイン取得音、影接触音（爆発）、ターボ音
- 全てWeb Audio API でプログラム生成（外部ファイル不要）

## スコアシステム
- 基本ポイント: 生存時間 × 10 (1秒ごと)
- コインボーナス: 1個 = +50点
- コンボ倍率: 影との距離 > 200pxを5秒維持で +1倍（最大5倍）
- ハイスコア保存: localStorage (`shadow-dodge-rush-highscore`)

## バイラル戦略（Grill Council結論）

### TOP3戦略
1. **死亡直前3秒の自動リプレイ画像/GIF生成** — リザルト画面に組み込み、ワンタップでSNSシェア（X/TikTok）。プレイヤーと影の軌跡を線で表示。
2. **TikTok/Shortsの縦型短尺動画でのフック投下** — 「あなたが追われているのは"0.5秒前のあなた"です」のキャッチで惹きつけ。
3. **最初の10秒で離脱させない設計** — チュートリアルは初回のみ、影は0.4倍速で開始、3秒で達成感小演出。
