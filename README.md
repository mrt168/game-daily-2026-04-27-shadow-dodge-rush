# シャドウダッシュ (Shadow Dash)

> 0.5秒前のあなたから逃げ続けろ。

自分の影が0.5秒遅れて追いかけてくるカジュアルアクションゲーム。スワイプで影から逃げ、生存時間でハイスコアを競え。

## 遊び方

1. 画面をスワイプしてキャラクターを動かす
2. 紫の影が0.5秒遅れて追いかけてくる
3. 影に触れたらゲームオーバー
4. 金のコインを取得してスコアアップ
5. 10秒後から赤い障害物が出現

## 技術

- Phaser 3.70（CDN）
- Vanilla JavaScript
- HTML5 Canvas
- localStorage（ハイスコア・チュートリアル状態）
- Web Audio API（プログラム生成サウンド）

## 開発

```bash
# ローカル起動
python3 -m http.server 8099
# → http://localhost:8099
```
